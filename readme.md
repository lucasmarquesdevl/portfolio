# 💻 Meu Portfólio Pessoal

Este é o repositório do meu portfólio profissional. Um site responsivo, moderno e multi-idioma desenvolvido para apresentar meus projetos, habilidades e certificações na área de TI e Desenvolvimento.

## 🚀 Tecnologias Utilizadas

* **HTML5** & **Tailwind CSS**: Estrutura e estilização moderna com foco em performance.
* **JavaScript (Vanilla)**: Lógica para troca de idiomas, manipulação de DOM e integração com WhatsApp.
* **Lucide Icons**: Conjunto de ícones leves e elegantes.
* **Google Fonts**: Tipografia focada em legibilidade (Inter).

## ✨ Funcionalidades

- 🌍 **Sistema Multi-idioma**: Suporte completo para Português (BR), Inglês (EN) e Espanhol (ES).
- 📱 **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- 💬 **Formulário Dinâmico**: Integração direta para envio de mensagens via API do WhatsApp.
- 📜 **Seção de Certificados**: Links diretos para comprovação de cursos e palestras.
- 🎨 **Efeitos Visuais**: Utilização de Glassmorphism e gradientes modernos.
